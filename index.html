18F accessibility dashboard

This is an early-stage prototype dashboard for 18F to track the accessibility characteristics across all its projects.

Notes:

  • This dashboard is accurate as of 4/10/2017.
  • For more details on how repositories are chosen, see the GitHub README.
  • The GitHub a11y issues count is found by searching for both open and closed issues in a project's repository matching the query accessibility OR a11y.
  • The aXe violations and aXe passes counts are found by visiting a project's homepage using Deque Systems' aXe-core tool. For more detailed information, please visit the project's homepage using the aXe Chrome plugin.

You can also download the information in this dashboard as CSV or JSON.

If you have any other questions or concerns, feel free to file a GitHub issue!
